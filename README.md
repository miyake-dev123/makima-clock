# Makima Clock Project

A character display clock project focused on embedded systems design, development environment configuration, and memory-constrained asset optimization.

## 🚀 Key Learning & Achievements

### 1. Embedded Systems Development
* **Hardware Configuration:** Configured development environments for **RP2040**, **FreeRTOS**, and **TFT_eSPI** to build a robust and responsive system architecture.
* **Environment Setup:** Gained hands-on experience in setting up and managing development workflows across both **Arduino IDE** and **PlatformIO**.

### 2. Hardware Resource & Memory Optimization
* **Analytical Problem Solving:** Analyzed hard memory and storage constraints inherent in microcontrollers to efficiently render visual assets.
* **Asset Trade-off Evaluation:** Evaluated efficiency and rendering trade-offs between utilizing **256-color GIF** files versus full **65536-color (16-bit RGB)** images for the character display, finding the optimal balance between visual quality and storage limits.

## 🛠️ Tech Stack & Keywords
* **Microcontroller:** RP2040
* **OS/Framework:** FreeRTOS
* **Display Library:** TFT_eSPI
* **IDE/Tools:** PlatformIO, Arduino IDE
* **Skills:** Memory Management, Image Optimization, Embedded Software Engineering
